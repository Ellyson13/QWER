	#HOW TO DIY
	
	
	
	
    

  • Kodi 17 Krypton download.
    • 

  • HOME Screen > Add-ons > Settings button  > Enable Unknown Sources.
    • 

  • BACK to HOME screen > Settings button  > File Manager  > Add Source.
    • 

  • Enter https://meserak86.github.io/QWER/ in the top box > Enter QWER in the bottom box > OK.
    • 

  • BACK to HOME screen > Add-Ons > Add-on Browser 
    • 

  • Install from zip file > QWER> plugin.program.QWERZ.zip.
    • 

  • Wait for Add-on enabled notification.
    • 

  • BACK to HOME screen > Add-ons > Program Add-ons > QWERwizard> Select Your build > Install
    • 

  • The build will now download, unzip
    • 

  • Wait for download > Force close > Power off if required(android)
    •
